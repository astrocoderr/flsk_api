# mock backend service for NutritionPro

### Installation
*node v7.6.0 or higher*
___
***Install dependencies***
```
yarn install
```

***On root (api) directory create .env file***
```
nano .env

```

***Pass MONGO_URI inside .env file***
```
MONGO_URI=mongodb://localhost:27017/fullstack_mock
```
